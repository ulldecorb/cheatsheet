# HTML Cheatsheet   

## TIPS
Edit Box:
```html
<div contenteditable="true">
```
Color Picker:
```html
<input type="color" value="#000">
```   
Progress Bar:
```html
<meter max="100" min="0">
<progress max="100" min="0">
```   
Spellcheck
```html
<input type="text" spellcheck="true">
```    
Quick Toggle
```html
<details>
  <summary>Title here</summary>
  The rest of toggled text...
</details>
```    
Spellcheck
```html
<input type="text" spellcheck="true">
```
