# EXPO
## COMMANDS
Start metro bundler
```
npx expo start    
```
Login
```
expo login -u YOUR-USERNAME-OR-EMAIL -p YOUR-PASSWORD
```
Emulator launcher
```
npx expo start --android

npx expo start --web

npx expo start --ios

```
```
a // android
w // web
i // ios

m // menu
r // reload
